# 🚀 Portfolio Games Collection

A collection of interactive web-based games built with HTML, CSS, and JavaScript. This repository showcases fundamental programming logic through classic arcade-style experiences.

## 🎮 Included Games

| Game | Description |
| :--- | :--- |
| **[Breakout](breakout.html)** | The classic brick-breaking game where you bounce a ball to destroy targets. |
| **[Snake](snaker.html)** | A timeless arcade experience: grow the snake while avoiding walls and your own tail. |
| **[Tic Tac Toe](tictactoe.html)** | A strategic two-player game of Xs and Os. |

## 🛠️ Technologies Used

*   **HTML5**: For structure and canvas rendering.
*   **CSS3**: For styling and layout.
*   **JavaScript (ES6+)**: For game logic, collision detection, and interactivity.

## 🚀 How to Run

Since these are pure client-side games, you don't need to install anything! 

1.  Clone this repository:
    ```bash
    git clone https://github.com/yourusername/portfolio.git
    ```
2.  Navigate to the folder.
3.  Open any `.html` file in your favorite web browser (Chrome, Firefox, Edge, etc.).

## 📝 Project Structure

*   `breakout.html`: Source code for Breakout game.
*   `snaker.html`: Source code for Snake game.
*   `tictactoe.html`: Source code for Tic Tac Toe game.
*   `skills.yaml`: Metadata regarding skills used in the project.
*   `prompts.md`: Documentation of prompts used during development.
